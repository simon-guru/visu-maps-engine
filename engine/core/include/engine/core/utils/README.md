# engine/core/include/engine/core/utils

Utilitários públicos compartilhados do core.

## Escopo desta fase

- modelagem de erro (`Result`, códigos de status, categorias de falha);
- logging básico com severidade e contexto;
- tipos de configuração base e helpers de validação.

## Não-escopo

- wrappers de bibliotecas específicas de plataforma;
- utilitários de módulos de domínio (render, tiles etc.).
