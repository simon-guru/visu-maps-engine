# apps

Aplicações host por plataforma que embutem o engine.

## Plataformas

- `android/`
- `ios/`
- `web/`
- `desktop-qt/`

## Funcionalidades foco nas apps host

- bootstrap do engine;
- ciclo de vida por plataforma;
- bridge de eventos de input/sensores;
- superfície de render integrada ao host.

## Status

Estruturas iniciais criadas para organizar integração por plataforma.
