# tools

Ferramentas auxiliares do ecossistema do engine.

## Módulos

- `asset_pipeline/`
- `shader_compiler/`
- `debug_tools/`

## Funcionalidades foco

- preparar assets para runtime;
- compilar/transpilar shaders por plataforma;
- facilitar diagnóstico técnico (frame/log/métricas).

## Objetivo

Padronizar fluxo de build, validação e inspeção técnica.
